üìñ Installation Steps

  1. Prerequisites
    Ensure Docker and Portainer are installed and running
  2. Open Portainer Settings
    Log into your Portainer web UI and navigate to Settings ‚Üí App Templates
  3. Add Template URL
    Paste the URL above into the URL field
  4. Save Settings
    Click Save settings to apply changes
  5. Browse Templates
    Go to App Templates in the sidebar to see all available templates
  6. Deploy!
    Select any template, configure options, and click Deploy the container
